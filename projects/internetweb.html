1. Internet.

- Is a global network of interconnected computers and servers that communicate using standard protocols like TCP or IP to share and acces information, resources, and lastly services.

2. Web.

- The Web is a World Wide Web is a system of interconnected documents, media and resources accesed via the internet using web browsers and identified by URLs.

3. How are the Internet and Web different?

- Internet is a Global Network of devices and systems it provides the infrastructure for communication and Data exchange, for examples like Email, Video Calls, and File Sharing. The Web relies on the Internet.

- A Web is a service that runs on the Internet that allows browsing and interacting with web pages and it browsing websites like an Wikipedia and YouTube. The Internet exists independently of the Web.

4. ASCII and Unicode.

- ASCII means American Standard Code for Information Interchange it is a character encoding standard that uses 7 bits to represent 128 characters, it includes English letters, Digits, and Basic Symbols.

- Unicode it is an universal character encoding standard that uses up to 32 bits to represent a range of characters from almost all languages, like scripts, and sysmbol worldwide.

5. What is UTF-8.

- Unicode Transformation Format - 8-bit in short UTF-8 it is a character encoding method for representing Unicode characters. It is a the most widely used encoding for the text on the web because of it's efficiency and compatibilty.

6. URL.

- It called Uniform Resources Locator, in short URL it is the address used to access a specific resource on the internet, such as a webpage, image, or file. it inlcudes the protocol. For example https://www.example.com/page1.

7. Domain Name.

- Is a human readable address used to identify a website or resources on the internet, making it easier to access than using an IP address.

8. HTTP.

- It is a Hypertext Transfer Protocol used for transfering data over the web. it defined how to request and responses are exchanged between a web browser and a web server and it enabling access to web pages and resources.

9. HTTPS.

- It is a Hypertext Transfer Protocol Secure it is an extension of HTTP that adds encryption using SSL/TLS, ensuring secure communication between a web browser and a server. it protect data from interception and tampering.

10. IP Address.

- It is Internet Protocol Address is a unique numerical label assigned to each device connected to a network, like the internet, it identifies the device and enables communication betwen devices over the network. There are two types: IPv4: A 32-bit address (192.168.0.1) & IPv6: A 128-bit address.

11. DNS.

- It is Domain Name System that translates human readable domain names into IP addresses that computers use to identify each other on the internet. it could acts like a phonebook for the internet.

12. W3C Recommendations.

Is a official standards and guidelines developed by the World Wide Web Consortium to ensure the interoperability and accesibility of web technologies.

13. XML, XHTML, HTML, HTML5.

- Extensible Markup Language (XML) A flexible markup language used to store and transport data in a structured format and it allows users to define custom tags.

- Extensible Hypertext Markup Language (XHTML) A stricter version of HTML that combines XML's syntax rules with HTML's structure and ensuring better error handling and compatibilty.

- Hypertext Markup Language (HTML) is a standard markup language used to create and structure content on the web it includes text, images, and links.

- HTML5 it is the latest version of HTML, which is includes a new features like multimedia support like audio and video. Better semantic elements, and it enchanced APIs for web development.
