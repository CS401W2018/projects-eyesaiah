Questions:

1. When do you use a table?

- Using a table for presenting structured data with rows and columns like schedules and data charts.

2. When do you not use a table?

- Do not use a table for layout or visual formatting and use CSS instead.

3. When do you need to do extra work for table accessibility?

- Do extra work for the accessibility when the tables are complex or contain nested data, using the headers, captions, and ARIA attributes.
