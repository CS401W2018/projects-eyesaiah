Questions:

1. Explain the difference between inline, embedded, and external CSS.

- Inline CSS it is applied directly with the element's "style" attribute.

- Embedded CSS it is defined with the a style tag inside the HTML head.

- External it is written in a separate .css file and linked using link.

2. Explain why you should only use external CSS.

- Using External Css that keeps HTML and CSS separate for better organization and it can be reused across multiple pages. It makes maintenance easier with one file. It speeds up page load times with browser caching. lastly it ensures consistent styling the across of the site.

3. What is a selector used for?

- The selector is used in the CSS to target HTML elements that you want to style. It also defines which elements on the page the styles that should apply to.
