Questions

1. What is the difference between block tags and inline tags?

- Block tags is a full-width and new line

- Inline tags is a inline and no line

2. What is the margin, border, and padding?

- Margin is a space outside the element between element and others.

- Border it is a edge or outline around the element.

- Padding is a space inside of the element between the content and the border.

3. What are units of measurement you can use in CSS?

- Abosulte (px, cm, mm, in, pt, pc).

- Relative (%, em, rem, vw, vh, vmin, vmax).

- Flexible (fr).

- Time (s, ms).

4. When would you use a figure tag?

- Use the < figure > tag group self contained content like images, diagrams, illustrations, code snippets, or other media, along with a optional < figcaption > for the caption or description.
